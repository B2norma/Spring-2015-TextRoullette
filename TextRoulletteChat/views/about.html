Welcome to the about page!

TextRoulete was created to provide users with a random chat room experience that has never before been in existence!
Below is a table of optional controls that can be entered into the text box.



Hyperlinks Insert any url into the text box
Images Insert the image url into the text box
YouTube Videos Insert the video url into the text box
Bold Font Begin with "//b "
Italic Font Begin with "//i "
Underlined Font Begin with "//u "
Highlighted Font Begin with "//mark "
Striked Font Begin with "//strike "
Small Font Begin with "//small "
Sized Font Begin with "//1 " - "//6 "
Superscript Begin with "//super "
Subscript Begin with "//sub "
Current Time Begin with "//time "
Close Tag Enter "///"



Users have the ability to enter their own user name before entering the chat realm as well as choose a personalized avatar from the provided list.
This list can be accessed by clicking the avatar logo in the Enter Chat Room page or by clicking the "Change Avatar" button inside any chat room.
The home page can be accessed at any time by clicking the TextRoulete icon at the top left corner of the page.
